# Sistema de Gestão de Vendas e Comissões

Sistema desenvolvido para gerenciamento de clientes, vendas e cálculo de comissões.

## Funcionalidades
- Cadastro de clientes
- Controle de vendas
- Cálculo automático de comissões
- Relatórios financeiros
- Histórico de operações

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Objetivo
Automatizar o controle comercial e financeiro, reduzindo processos manuais.
